# The Art of Noticing

A minimalist photography gallery focused on observing the ordinary and finding meaning in overlooked moments.

This project is a quiet visual archive light, texture, form, and everyday scenes, presented with simplicity and intention.

## About

- Built using **HTML & CSS**
- Masonry-style layout using modern CSS
- Responsive across desktop and mobile
- No frameworks, no JavaScript, just the essentials


## Live Site

[*Gallery*](https://sotooba.github.io/My-Gallery/)


## Philosophy

> Photography is the art of noticing what usually goes unnoticed.

This site exists to slow down the scroll and invite attention.
